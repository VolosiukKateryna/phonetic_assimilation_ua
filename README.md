Цей проєкт надає інтерактивний веб‑інтерфейс для демонстрації складних лінгвістичних правил, що регулюють вимову українських слів.
Основна увага приділяється:
- Асиміляції за дзвінкістю та глухістю – автоматичне перетворення приголосних залежно від сусідніх звуків.
- Пом’якшенню та напівпом’якшенню приголосних – відображення м’яких варіантів у різних позиціях слова.
Особливості
- Інтерфейс реалізовано у Gradio, що дозволяє працювати напряму у браузері.
- Зліва відображаються зміни у вимові слів за глухістю та дзвінкістю.
- Справа – результати аналізу м’якості та напівпом’якшення.
